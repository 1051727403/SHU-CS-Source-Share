# 实用计算机英语                                                    

简介：这门课作为一门4分的选修课，是在第十周提前考的，考试方式为闭卷

优点：不需要制作项目，分值高，管得不严，适合划水

缺点：课后作业习题较多(当前目录下仅为其中的一小部分)，最后个人感觉很难给高分

总结：如果不对绩点有高需求，又想划水，又想给分多，并且自身有一定英语水平的可以考虑选
